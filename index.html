Encriptador - Alura Latam

Este proyecto es un encriptador de texto interactivo que permite a los usuarios escribir cualquier mensaje y recibirlo encriptado. Utilizando HTML para la estructura, CSS para un diseño atractivo y JavaScript para la funcionalidad, el encriptador convierte el texto ingresado en un formato encriptado al instante.

Reducir la mortalidad por tráfico en EE. UU.

En este proyecto, he desarrollado una estrategia integral para abordar el grave problema de las muertes relacionadas con el tráfico en EE. UU. A través de un enfoque metódico, importé, limpié y manipulé un conjunto de datos relevante, seguido de una cuidadosa visualización para identificar patrones críticos.

Dashboard

En este proyecto, he desarrollado un panel interactivo de análisis de ventas utilizando Power BI, una herramienta poderosa de visualización de datos. Mi objetivo fue transformar datos complejos en información visual comprensible para ayudar a la toma de decisiones estratégicas. Importé datos de diferentes fuentes, realicé la limpieza y modelado necesarios, y diseñé visualizaciones atractivas que muestran tendencias de ventas, comparativas entre productos y rendimiento por región.

Análisis y previsión de series temporales con Python

En este proyecto, he aplicado el modelo ARIMA para la previsión de ventas de muebles y material de oficina. A través de un enfoque analítico, he comparado los resultados obtenidos y los he visualizado para entender mejor las tendencias en las ventas.
